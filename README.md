# CSCPay app
This app is meant for demonstration purposes only.





## Installation


```bash
  git clone https://github.com/Shakira4242/cscpay_app.git
  cd cscpay_app
  npm install
  
```
    
## Tech Stack

**App:** React Native, Material UI, react-native-ble-manager

**Server:** Node, Express
